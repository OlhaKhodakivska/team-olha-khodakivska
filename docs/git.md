# Git Guide

## Clone repository

```bash
git clone <repository-url>
```

## Check remotes

```bash
git remote -v
```

## Add upstream

```bash
git remote add upstream <repository-url>
```

## Create new branch

```bash
git checkout -b feature-name
```

## Add changes

```bash
git add .
```

## Commit

```bash
git commit -m "your message"
```

## Push to origin

```bash
git push origin branch-name
```

## Create Pull Request

Go to GitHub and create a PR from your fork to upstream repository.
